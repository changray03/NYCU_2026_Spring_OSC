# 作業系統總整與實作 (OSC) - LAB
這裡是我在國立陽明交通大學學習「作業系統總整與實作」的 Lab。主要內容為 Bare-metal Programming，從頭建立一個簡易的 OS kernel。

## Lab 1
* UART Driver 開發
* OpenSBI interface
* Memory layout using linker script
* BSS clear, SP pointer setup using Assembly

## Lab 2
* 動態載入 kernel image by UART Bootloader
* Device tree parsing
* initrd.cpio parsing
